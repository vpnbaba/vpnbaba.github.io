前期准备：获取必要工具

由于App Store的地区限制政策，中国大陆区的App Store可能无法直接搜索到小火箭(Shadowrocket)。这时您需要：

  1. 注册一个非中国大陆区的Apple ID（如美国、香港等）
  2. 登录该Apple ID后，在App Store中搜索"Shadowrocket"
  3. 下载并安装应用（小火箭为付费应用，价格通常在2.99美元左右）

值得注意的是，小火箭虽然需要一次性付费购买，但其稳定性和易用性绝对物超所值。相比那些所谓的"免费"VPN应用，小火箭不会收集用户数据，也没有烦人的广告，更不会突然停止服务。

详细配置指南：从零开始设置

V2ray节点信息的解读

一个完整的V2ray节点通常包含以下关键信息：

  • 服务器地址(Address)：可以是IP地址或域名
  • 端口(Port)：连接服务器使用的端口号
  • 用户ID(UUID)：用于身份验证的唯一标识符
  • 额外ID(AlterId)：增强安全性的可选参数
  • 加密方式(Security)：如auto、aes-128-gcm等
  • 传输协议(Network)：通常为tcp、ws(WebSocket)等
  • 伪装类型(Type)：none、http、srtp等
  • 伪装域名(Host)：用于TLS验证的域名
  • 传输层安全(TLS)：是否启用TLS加密

理解这些参数的含义对后续配置和故障排除都大有裨益。

小火箭中导入V2ray节点

导入节点是小火箭配置的核心步骤，具体操作如下：

  1. 打开小火箭应用，点击右上角的"+"按钮
  2. 选择"手动输入"或"扫描二维码"（如果节点提供者给出了二维码）
  3. 根据节点信息填写各项参数：
    • 类型选择"Vmess"
    • 填入服务器地址和端口
    • 用户ID填写UUID
    • 加密方式通常选择"auto"
    • 根据节点信息选择传输协议和伪装类型
  4. 点击"完成"保存配置

对于新手来说，最简便的方式是使用二维码导入。许多V2ray服务商都提供节点二维码，只需在小火箭中点击"扫描二维码"按钮，对准二维码即可自动完成所有配置。

连接测试与优化

完成节点导入后，建议进行以下操作：

  1. 返回小火箭主界面，选择刚添加的节点
  2. 开启顶部开关尝试连接
  3. 观察连接状态指示灯：
    • 绿色表示连接成功
    • 黄色表示正在连接
    • 红色表示连接失败
  4. 连接成功后，访问ip.sb等网站检查IP是否已变更

如果连接失败，可以尝试以下排查步骤： - 检查节点信息是否输入正确，特别是UUID和端口 - 尝试切换不同的网络（WiFi/移动数据） - 联系节点提供商确认服务状态 - 尝试更换其他节点

高级配置技巧

分流规则设置

小火箭强大的分流功能可以让您灵活控制哪些流量走代理：

  1. 进入"配置"→"分流规则"
  2. 选择"添加规则"可以创建自定义规则
  3. 常见的规则类型包括：
    • 域名后缀规则：如"*.google.com"
    • IP范围规则：如"8.8.8.8/24"
    • GEOIP规则：按国家地区分流
  4. 设置规则动作：代理、直连或阻止

合理的分流设置不仅能提升访问速度，还能节省流量。例如，您可以将国内网站设为直连，仅对境外资源使用代理。

订阅功能的使用

如果您使用的是付费V2ray服务，很可能支持订阅链接：

  1. 在小火箭中点击"订阅"
  2. 选择"添加订阅"
  3. 填入订阅URL和备注名称
  4. 设置自动更新频率（建议每天）
  5. 保存后点击"更新"获取节点列表

订阅功能的最大优势是节点更新方便。当服务商更换节点时，您只需在小火箭中点击"更新"即可获取最新节点，无需手动一个个添加。

安全与隐私保护建议

科学上网虽好，但安全更为重要。以下是一些关键建议：

  1. 选择支持TLS加密的V2ray节点
  2. 定期更换UUID（特别是使用订阅服务时）
  3. 避免在代理环境下登录重要账户
  4. 不要使用来源不明的免费节点
  5. 考虑开启小火箭的"增强模式"（需要安装配置文件）

常见问题解决方案

连接速度慢怎么办？

网络速度受多种因素影响，可以尝试：

  1. 更换其他节点（选择物理距离更近的服务器）
  2. 调整传输协议（如从TCP改为WebSocket）
  3. 检查本地网络状况
  4. 联系服务商确认服务器负载情况

小火箭频繁断开连接

稳定性问题可能由以下原因导致：

  1. 网络环境不稳定（尝试切换WiFi/4G/5G）
  2. 节点被封锁（尝试更换端口或传输协议）
  3. 手机系统限制（检查后台刷新权限）
  4. 小火箭版本过旧（更新到最新版本）

如何确认代理是否生效

有几个简单的方法：

  1. 访问ip.sb或whatismyip.com查看IP地址
  2. 尝试访问之前无法打开的网站
  3. 使用小火箭的"连接检查"功能
  4. 观察浏览器访问网站时的速度变化

FAQ

QuantumultX 如何实现按应用分流？
QuantumultX 可以根据不同应用程序的流量类型设置分流规则，通过策略组选择最优节点。这样可保证游戏、视频和浏览器流量经过最优节点，提高访问速度和网络稳定性。
Mihomo 启动后 CPU 占用高，有哪些优化建议？
检查是否启用了大量日志或复杂脚本，关闭不必要的规则提供者与日志调试；在多核服务器上启用并发或调整线程数；更新到最新版本以获得性能优化，并监控是否有异常连接导致处理压力。
Clash 的策略组可设置延迟阈值吗？
可以。用户可定义延迟阈值，当节点延迟超过设定值时，策略组会自动切换至其他低延迟节点，提高网络速度和稳定性，优化访问体验。
如何备份 Clash 配置文件？
Clash 的配置文件通常位于程序目录或配置文件夹中（如 Windows 下的 .config/clash）。用户可直接复制该 YAML 文件以备份，也可将订阅链接保存，便于后续恢复或在其他设备上同步使用。
V2Ray 的路由规则如何实现国内直连？
通过配置域名列表或 IP 段，将国内流量指向直连出口，而国外流量走代理节点。结合策略组和定期更新规则，可保持国内访问速度，同时优化海外访问体验。
Hiddify Next 如何管理用户流量配额？
管理员可以为每个用户设置上传和下载带宽以及总流量上限。超出限制可自动降速或阻止访问，保证服务器资源合理分配，提高整体网络公平性和用户体验。
Shadowrocket 的测速功能如何优化节点选择？
Shadowrocket 可测量节点延迟和丢包率，并根据结果排序策略组节点。延迟低的节点会优先使用，保证网页浏览、视频和游戏的速度和稳定性，提高整体网络体验。
Surfboard 是否支持 Clash 配置？
Surfboard 是一款支持多种协议的客户端应用，它能够兼容 Clash 的配置文件格式，用户只需导入 YAML 文件即可。该应用以轻量、高速著称，常用于 iOS 和 macOS 系统。
Trojan-Go 的 MUX 功能能降低延迟吗？
启用 MUX 后，多条 TCP/QUIC 连接可以共享单个隧道，减少握手次数和连接延迟。对于频繁短连接应用如游戏或社交软件，可以明显提升响应速度。
Shadowrocket 支持哪几种代理协议？
Shadowrocket 支持多种协议，包括 Shadowsocks、Vmess、VLess、Trojan、HTTP、SOCKS5 等。它还可解析部分 QuantumultX 与 Clash 的订阅格式，适合 iOS 用户进行全协议代理管理与切换。
